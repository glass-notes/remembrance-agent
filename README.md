# remembrance-agent
Java package for Remembrance Agents! Based on Rhodes/Starner

## Commands

### Building

```bash
./gradlew assemble
```

### Running

#### Running for the first time

```bash
java -jar build/libs/remembrance-agent-v1.0-all.jar
```

#### Running from cache

```bash
java -jar build/libs/remembrance-agent-v1.0-all.jar --use-cache
```
